List of Commands in Azure Databricks
======================================
dbutils.fs.mkdirs(<path>) --- make a directory with the provided path
dbutils.fs.ls(<path>) -- list of files in a path
dbutils.fs.mv(<source>, <dest> ) -- move files from source to destination
dbutils.fs.cp(<source>, <dest>) -- copy files
dbutils.fs.rm(<path>) -- remove files

isinstance(<var1>, <DataType>) -- check if something is of defined datatype, can be used to check if it is Koalas Dataframe
spark.read.format(<format>).option("header", <bool>).option("delimiter", <delimiter>).load(<path>) -- reads csv file from a given location
koalas.read_excel(x.path,dtype = dtype, sheet_name = sheet).to_spark() -- read excel using koalas and convert to spark dataframe
